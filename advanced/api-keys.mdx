### OpenAI APIキー

Cursorでは、独自のOpenAI APIキーを入力することで、自己負担で好きなだけAIメッセージを送信することができます。

APIキーは[こちら](https://platform.openai.com/account/api-keys)から取得できます。独自のAPIキーを使用するには、`Cursor Settings` > `Models` > `OpenAI API Key`に移動し、APIキーを入力します。その後、「検証」ボタンをクリックします。キーが検証されると、OpenAI APIキーが有効になります。

### Anthropic APIキー

OpenAIと同様に、独自のAnthropic APIキーを設定することで、自己負担でClaude系のモデルを使用することができます。

### Google APIキー

Google APIキーについては、独自のAPIキーを設定することで、自己負担で`gemini-1.5-flash-500k`などのGoogleモデルを使用することができます。

### Azure連携

最後に、独自のAzure APIキーを設定することで、自己負担でAzure OpenAIモデルを使用することもできます。

### APIキーは保存されたり、デバイスから送信されたりしますか？

APIキーは保存されませんが、リクエストごとに当社のサーバーに送信されます。最終的なプロンプトの構築は当社のバックエンドで行われるため、すべてのリクエストは当社のバックエンドを経由します。
