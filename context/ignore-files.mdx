---
title: 除外ファイル
---

Cursorの機能（[コードベースのインデックス作成](/context/codebase-indexing) など）から特定のファイルを除外するには、
プロジェクトのルートディレクトリに`.cursorignore`ファイルを作成します。これはgitの`.gitignore`と同じように動作します。

`.cursorignore`は`.gitignore`の設定を尊重します。既に`.gitignore`がある場合、そのファイルは自動的に除外されます。
追加で除外したいファイルがある場合は、`.cursorignore`ファイルに追記することができます。

### `.cursorignore` ファイルの例

```
# `dist` ディレクトリ内のすべてのファイルを除外
dist/

# すべての `.log` ファイルを除外
*.log

# 特定のファイル `config.json` を除外
config.json
```
